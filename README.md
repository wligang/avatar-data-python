# avatar-data-python
avatar项目使用python爬取csdn的数据的模块


## 说明
 avatar-data-python工程是一个可独立运行在windos或者Linux上的一个python爬虫工具，在本项目中，主要目的是为了爬去CSDN的用户数据

在本项目中，拥有主动和被动来爬取用户的数据，如果你是老炮，那可能怎么运行您已经知道，后面抽空我会为菜鸟准备一个使用说明
